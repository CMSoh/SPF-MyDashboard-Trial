# SPF One-Front-Door E-Services Portal

## Overview

This project is a React-based web application that serves as a centralized portal for Singapore Police Force (SPF) e-services. It provides a user-friendly interface for citizens to access various services, track applications, and report incidents.

## Features

*   **Dashboard:** A personalized overview of active items, required actions, and recent updates.
*   **Track Status:** A comprehensive table to track the real-time progress of applications and incident reports.
*   **Report Incident:** A form for submitting non-emergency incident reports.
*   **Apply for Services:** A searchable directory of all available e-services.
*   **Responsive Design:** The application is designed to be fully responsive and accessible on both desktop and mobile devices.
*   **Modern UI/UX:** The user interface is built with a focus on clarity, ease of use, and a visually appealing aesthetic.

## Tech Stack

*   **React:** A JavaScript library for building user interfaces.
*   **TypeScript:** A typed superset of JavaScript that compiles to plain JavaScript.
*   **Tailwind CSS:** A utility-first CSS framework for rapid UI development.
*   **React Router:** A declarative routing library for React.
*   **Vite:** A fast and lightweight build tool for modern web projects.
*   **ESLint:** A pluggable and configurable linter tool for identifying and reporting on patterns in JavaScript.
